# 高级软件工程师/系统架构设计师备考网站规格文档

## 1. 项目概述

- **项目名称**: 架构师备考通
- **项目类型**: 移动端优先的Web应用
- **核心功能**: 按天计划学习，知识点图文展示，支持每日复习提醒
- **目标用户**: 备考高级软件工程师/系统架构设计师的考生

## 2. UI/UX 规格

### 布局结构

- **移动端优先设计**: 最大宽度 480px，居中显示
- **页面结构**:
  - 顶部导航栏 (固定)
  - 主内容区域 (可滚动)
  - 底部Tab导航 (首页/学习/我的)

### 视觉设计

- **色彩方案**:
  - 主色: #1E3A5F (深蓝色 - 专业感)
  - 辅色: #3B82F6 (亮蓝色 - 活力)
  - 强调色: #F59E0B (橙色 - 重点提示)
  - 背景: #F8FAFC (浅灰白)
  - 卡片背景: #FFFFFF
  - 文字主色: #1F2937
  - 文字辅色: #6B7280

- **字体**:
  - 主字体: "PingFang SC", "Helvetica Neue", sans-serif
  - 标题: 18px (粗体)
  - 正文: 14px
  - 辅助: 12px

- **间距系统**:
  - 页面边距: 16px
  - 卡片内边距: 16px
  - 元素间距: 12px

- **动效**:
  - 页面切换: 滑动动画 300ms
  - 按钮点击: scale(0.98) 反馈
  - 卡片进入: 淡入上移

### 组件列表

1. **DayCard (日期卡片)**: 显示第N天、日期、知识点数量、完成状态
2. **KnowledgeCard (知识点卡片)**: 标题、摘要、标签、进度状态
3. **KnowledgeDetail (知识点详情)**: 图文内容、重点标记、记忆技巧
4. **ProgressBar (进度条)**: 显示整体学习进度
5. **TabBar (底部导航)**: 三个Tab切换

## 3. 功能规格

### 页面结构

1. **首页** (/):
   - 显示总学习天数计划 (约100天)
   - 进度条展示总体完成度
   - 快速进入今日学习

2. **学习计划页** (/plan):
   - 按日期排列的学习计划列表
   - 每天的知识点数量和完成状态
   - 点击进入当天知识点列表

3. **知识点列表页** (/day/:dayId):
   - 当天所有知识点的卡片列表
   - 显示已复习/未复习状态
   - 点击进入知识点详情

4. **知识点详情页** (/knowledge/:id):
   - 完整的图文内容
   - 重点难点标记
   - 记忆技巧/口诀
   - 返回按钮

5. **我的** (/profile):
   - 学习统计
   - 设置每日提醒

### 数据模型

```typescript
interface DayPlan {
  id: number;
  date: string;
  title: string;
  knowledgeCount: number;
  completedCount: number;
}

interface Knowledge {
  id: number;
  dayId: number;
  title: string;
  summary: string;
  content: string; // Markdown格式，支持图文
  tags: string[];
  difficulty: 'easy' | 'medium' | 'hard';
  isReviewed: boolean;
}
```

## 4. 技术栈

- **前端**: React + Vite + React Router
- **样式**: CSS Modules / Styled Components
- **后端**: Node.js + Express (简单API)
- **数据**: JSON文件存储 (便于部署)
